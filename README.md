# Employees CRUD REST API

This is a REST API to create, read, update and delete employee records. There are five endpoints.
1. Post - to create employee records
1. Get all - Fetches all employee records
1. Get one by ID - fetches on employee record on Id
1. Update-  updates employee record on Id
1. Delete- deletes employee record on Id

## Deployment
Api has been deployed on [Render](https://render.com/) and can be accessed on url https://employee-crud-plia.onrender.com/api
